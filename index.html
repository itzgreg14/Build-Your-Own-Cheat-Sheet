CSS Font Cheat Sheet

CSS Font Properties

Property Value Description
font-size; 5px, 5em, 5rem This propert dictates the size of the font.
font-family; Georgia, 'sans-serif', cursive, etc. This property dictates the style of the font.
font-weight; normal, bold, lighter, bolder, 100-900 This property sets the weight of the font.
font-style; normal, italix, oblique This property dictated how the font will be styled.
